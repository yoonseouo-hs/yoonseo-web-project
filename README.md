## 📖 프로젝트 설명

`project_personal`은 웹 프로그래밍 기초를 바탕으로 **HTML, CSS, JavaScript**를 사용해 제작한 프로젝트입니다.  
10~30대 남녀를 대상으로 다양한 코디를 모아보고 참고할 수 있는 웹사이트입니다.
---

## ⚙️ 주요 기능

- 🏠 **홈(Home)** 
- 👤 **소개(About)** 
- 📂 **프로젝트(Project)**  
- ✉️ **연락(Contact)**   
- 🎯 **CTA 버튼(Call To Action)** 
---

## 🧰 사용 기술

| 구분 | 기술 스택 |
|------|------------|
| **Frontend** | HTML5, CSS3, JavaScript (Vanilla JS) |
| **Design** | Figma (와이어프레임, 사이트맵 설계) |
| **Version Control** | Git, GitHub |
| **Deployment** | GitHub Pages |
